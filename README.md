#  Restaurant Reservation System (MERN)

A full-stack web application for managing restaurant table reservations using the MERN stack.

##  Features
- User authentication (Login/Signup)
- Table reservation system
- Admin dashboard for managing bookings
- Responsive UI

##  Tech Stack
- Frontend: React.js, Vite
- Backend: Node.js, Express.js
- Database: MongoDB

##  Getting Started

### 1. Clone the repository
git clone https://github.com/Simran-sinha0910/restaurant-reservation-mern.git

### 2. Install dependencies
cd backend
npm install

cd ../frontend
npm install

### 3. Run the project
# Backend
cd backend
npm run dev

# Frontend
cd frontend
npm run dev

##  Project Structure
- frontend/ → React application  
- backend/ → API and server
